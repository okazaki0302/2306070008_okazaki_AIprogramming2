# 📚 Wikipedia要約アプリ

このアプリは、ユーザーが入力したキーワードに基づいてWikipediaの該当ページを取得し、要約を表示するWebアプリです。Streamlitを使って構築されています。

## 🔧 使用技術
- Streamlit
- Wikipedia API（認証不要）
- Python

## 📥 入力
- キーワード（テキスト）

## 📤 出力
- Wikipediaページのタイトル
- 要約文（最大5文）
- 該当ページへのリンク

## 📁 ファイル構成
- `app.py`: Streamlit UI
- `logic.py`: Wikipedia API呼び出しと要約処理
